Class Overview
==============
The ProdottoService class provides a service layer for retrieving product information. It encapsulates interactions with the ProdottoDAO to fetch product details by ID.

Key State & Dependencies:
- ProdottoDAO: Handles the database operations for retrieving product data.

Constructor: ProdottoService(ProdottoDAO)
=========================================
Parameters:
- prodottoDAO (ProdottoDAO): The data access object for products.

Categories and Choices:
-----------------------
1. prodottoDAO:
   - Null
   - Non-null (Valid object)

Test Frames:
------------
ID  | Choices                   | Expected Behavior
--- | ------------------------- | -----------------
C1  | prodottoDAO=Null          | Throw IllegalArgumentException("ProdottoDAO cannot be null")
C2  | prodottoDAO=Non-null      | Same instance assigned to internal field; Object created successfully

Suggested JUnit 5 Tests:
------------------------
1. shouldThrowExceptionWhenProdottoDAOIsNull
   - Arrange: ProdottoDAO dao = null;
   - Act: new ProdottoService(dao);
   - Assert: assertThrows(IllegalArgumentException.class);

2. shouldInitializeSuccessfullyWhenProdottoDAOIsValid
   - Arrange: ProdottoDAO dao = mock(ProdottoDAO.class);
   - Act: ProdottoService service = new ProdottoService(dao);
   - Assert: assertNotNull(service);

Method: getProdottoById(int id)
===============================
Parameters:
- id (int): The unique identifier of the product to retrieve.

State:
- Database content (via ProdottoDAO).

Categories and Choices:
-----------------------
1. id:
   - <= 0 (Invalid, e.g., 0, -1)
   - > 0 (Valid, e.g., 1, 100)

2. DAO Interaction (prodottoDAO.getProdottoById):
   - Returns Prodotto (Product found)
   - Returns null (Product not found)
   - Throws SQLException (Database error)

Test Frames:
------------
ID  | Choices                                | Expected Behavior
--- | ---------------------------------------| -----------------
G1  | id=0                                   | Throw IllegalArgumentException("Product ID must be greater than 0")
G2  | id=-1                                  | Throw IllegalArgumentException("Product ID must be greater than 0")
G3  | id=1, Product Found                    | Call DAO, Return Prodotto object
G4  | id=1, Product Not Found                | Call DAO, Return null
G5  | id=1, DAO throws SQLException          | Throw SQLException

Suggested JUnit 5 Tests:
------------------------
1. shouldThrowExceptionWhenIdIsInvalid
   - Parameterized Test (ValueSource): ints {0, -1, -100}
   - Arrange: ProdottoService service = new ProdottoService(mockDao);
   - Act: service.getProdottoById(invalidId);
   - Assert: assertThrows(IllegalArgumentException.class);

2. shouldReturnProductWhenFound
   - Arrange: Mock dao to return specific Prodotto when getProdottoById(1) is called.
   - Act: service.getProdottoById(1);
   - Assert: assertEquals(expectedProdotto, result); verify(dao).getProdottoById(1);

3. shouldReturnNullWhenProductNotFound
   - Arrange: Mock dao to return null when getProdottoById(1) is called.
   - Act: service.getProdottoById(1);
   - Assert: assertNull(result); verify(dao).getProdottoById(1);

4. shouldPropagateSQLException
   - Arrange: Mock dao to throw SQLException when getProdottoById(1) is called.
   - Act: service.getProdottoById(1);
   - Assert: assertThrows(SQLException.class);
