Class Overview
==============
The LogoutService class is responsible for managing user logout operations. Its sole responsibility is to securely terminate a user's session.
Key state variables: None (Stateless service).

Method: logout(HttpSession session)
===================================
Parameters:
- session: The HttpSession object representing the current user session.

State:
- The state of the HttpSession object (active vs invalid).

Categories and Choices:
-----------------------
- Session Object:
  - Null
  - Non-null instance

- Session State (if Non-null):
  - Active (Valid)
  - Already Invalidated

Test Frames:
------------
TF1: Session is null
   - Input: session = null
   - Expected: IllegalArgumentException ("Session cannot be null when performing logout.")

TF2: Session is active
   - Input: session = [Valid Active Mock Session]
   - Expected: session.invalidate() is called.

TF3: Session is already invalidated
   - Input: session = [Invalidated Mock Session]
   - Expected: IllegalStateException (propagated from session.invalidate())

Suggested JUnit 5 Tests:
------------------------
@Test
void shouldThrowExceptionWhenSessionIsNull() {
    // Arrange
    LogoutService service = new LogoutService();
    // Act & Assert
    IllegalArgumentException exception = assertThrows(IllegalArgumentException.class, () -> service.logout(null));
    assertEquals("Session cannot be null when performing logout.", exception.getMessage());
}

@Test
void shouldInvalidateSessionWhenSessionIsActive() {
    // Arrange
    LogoutService service = new LogoutService();
    HttpSession mockSession = mock(HttpSession.class);
    // Act
    service.logout(mockSession);
    // Assert
    verify(mockSession).invalidate();
}

@Test
void shouldThrowExceptionWhenSessionIsAlreadyInvalidated() {
    // Arrange
    LogoutService service = new LogoutService();
    HttpSession mockSession = mock(HttpSession.class);
    doThrow(new IllegalStateException("Session already invalidated")).when(mockSession).invalidate();
    
    // Act & Assert
    assertThrows(IllegalStateException.class, () -> service.logout(mockSession));
}
