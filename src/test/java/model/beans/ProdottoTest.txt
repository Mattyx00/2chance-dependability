Class Overview
The Prodotto class represents a product entity within the e-commerce system. It serves as a data transfer object (DTO) holding product details such as unique identifier, quantity, price, weight, dimensions, brand, model, image, description, and associations to a category, reviews, and specifications. Key validation logic is enforced in setter methods to ensure data integrity: numerical values (quantity, price, weight) must be non-negative, and essential string fields (brand, model, description) as well as object references (category, lists) must not be null or empty.
