### 1. CLASS OVERVIEW

**Class Name**: Specifiche
**Package**: model.beans

**Description**:
The Specifiche class represents a product specification (technical detail) in an e-commerce system. It encapsulates a key-value pair where 'nome' is the specification name (e.g., "RAM", "Processor") and 'valore' is its value (e.g., "16GB", "Intel i7").

**Important Fields**:
- nome: String - The specification name/key
- valore: String - The specification value

**Invariants** (from JML):
- invariant nome != null && !nome.isEmpty()
- invariant valore != null && !valore.isEmpty()

Both fields must be non-null and non-empty, ensuring every specification is complete and meaningful.
