### 1. CLASS OVERVIEW

**Class Name**: Prodotto
**Package**: model.beans

**Description**:
Prodotto represents a product in an e-commerce catalog. It contains product details (marca, modello, descrizione, prezzo, etc.), associated specifications, and customer reviews.

**Important Fields**:
- id: int - Product identifier
- marca, modello, descrizione: String - Product descriptions
- dimensioni, immagine: String - Product attributes
- prezzo: double - Product price
- peso: double - Product weight
- quantitaProdotto: int - Available stock quantity
- categoria: Categoria - Product category
- specifiche: ArrayList<Specifiche> - Technical specifications
- recensioni: ArrayList<Recensione> - Customer reviews

**Invariants** (from JML):
- invariant prezzo >= 0.0
- invariant peso >= 0.0
- invariant quantitaProdotto >= 0
