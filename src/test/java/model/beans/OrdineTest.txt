### 1. CLASS OVERVIEW

**Class Name**: Ordine
**Package**: model.beans

**Description**:
Ordine represents a customer order in an e-commerce system. It contains order details including date, shipping address, total price, the customer (Utente), and the shopping cart (Carrello) with all ordered items.

**Important Fields**:
- id: int - Order identifier
- dataOrdine: Date - Order placement date
- indirizzo: String - Shipping address
- prezzoTotale: double - Total order price
- utente: Utente - Customer placing the order  
- carrello: Carrello - Shopping cart with order items

**Invariants** (from JML):
- invariant prezzoTotale >= 0.0
