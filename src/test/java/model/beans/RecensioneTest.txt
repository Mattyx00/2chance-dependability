Class Overview
The Recensione class represents a review entity. Its primary responsibility is to hold data details about a review, including the user who made it, the product being reviewed, a numerical rating, text content, and the date.
Key state variables:
- id: Integer identifier (no specific validation logic shown in code, default int behavior).
- valutazione: Integer rating, strictly constrained between 1 and 5.
- dataRecensione: Date object, must not be null.
- testo: String content, must not be null or empty/blank.
- utente: Utente object involved, must not be null.
- prodotto: Prodotto object involved, must not be null.

Input validation is enforced in the full constructor and individual setters, throwing IllegalArgumentException for violations.
