Class Overview
The Utente class represents a registered user in the system. It maintains personal information (name, surname, email, phone), authentication details (password), system roles (admin status), and references to related entities (orders, reviews).
Key state variables:
- id: Distinct integer identifier.
- nome, cognome, email, telefono: User's personal details (cannot be null/empty).
- password: User's hashed password (cannot be null/empty, stored as SHA-1 hash).
- ordini: List of orders placed by the user.
- recensioni: List of reviews written by the user.
- admin: Boolean flag for administrative privileges.
