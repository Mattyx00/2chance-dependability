Class Overview
The Specifiche class represents a product specification consisting of a name (nome) and a value (valore).
Key state variables:
- nome: String (enforced to be non-null and non-empty/non-blank in validated methods)
- valore: String (enforced to be non-null and non-empty/non-blank in validated methods)
The class provides a no-argument constructor (which initializes fields to default null) and a validated parameterized constructor that enforces state invariants.
